Raw Flattened Metadata type.

All keys are converted to string Any keys that are arrays are flattened, undefined keys are removed

Type Parameters

  • T extends IaBaseMetadataType

Example

type x = RawFlattenedMetadata<{ a: "a", b: ["c", "d", "e"], c:undefined }>;
// Results in the following type
{
 a: "a";
 b: "c";
 "b[1]": "d";
 "b[2]": "e";
}
