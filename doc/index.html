internetarchive-ts - v0.1.0

Internet Archive Typescript API client

This is a work-in-progress verion and not fully functional.

This API is intended to serve as a base for server-, desktop- and web-applications and offers both ways to easily fetch, create and update items from the Internet Archive as well as access the full functionality of the Archive.org API. The code is partially based on the Internet Archive Python API client and contains all features of the python package, however no cli is included. For an out-of-the-box cli tool, please use the python package.
